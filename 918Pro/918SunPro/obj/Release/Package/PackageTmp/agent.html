佣金提成计算:代理佣金 = 纯利润 × 当月佣金比例

举例3假若您介绍的客户在该月为我们带来 900,000元人民币的纯利，同时您的活跃会员人数是5至9位之间，即符合了第四级的当月总纯利收入和第一级的每月最低活跃会员，那么该月的结算则会按照第一级的30% 提成去计算。

举例4假若您介绍的客户在该月为我们带来 9,000元人民币的纯利, 同时您的活跃会员人数是50位或以上，即符合了第一级的当月总纯利收入和第五级的每月最低活跃会员，那么该月的结算则会按照第一级的30% 提成计算。
